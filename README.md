# ICPC Top 3 Colombia

Sistema personal de entrenamiento orientado a alcanzar el top 3 de la
Maratón Nacional de Programación de Colombia antes de finalizar la carrera.

## Objetivo principal

Construir el nivel individual, técnico y colectivo necesario para disputar
un puesto entre los tres mejores equipos de Colombia.

## Etapas

- 2026: fundamentos, diagnóstico competitivo y construcción del entorno.
- 2027: consolidación del equipo y entrada al top nacional.
- 2028: preparación específica para disputar el top 3.

## Estructura

- `contests/`: concursos individuales y simulaciones de equipo.
- `problems/`: soluciones individuales fuera de concursos.
- `library/`: implementaciones propias y verificadas.
- `notes/`: teoría, editoriales propios y registro de errores.
- `roadmap/`: planificación, diagramas y línea de tiempo.
- `data/`: métricas y registros estructurados.
- `tools/`: generadores, stress testing y automatizaciones.

## Principios

1. No copiar soluciones sin comprenderlas.
2. Registrar la ayuda utilizada en cada problema.
3. Hacer upsolving después de cada concurso.
4. Verificar las implementaciones antes de agregarlas a la biblioteca.
5. Medir progreso mediante desempeño, no solo cantidad de problemas.
6. No almacenar libros ni material comercial en este repositorio.
